¿Qué es SSH?

SSH (Secure Shell) es un protocolo de red criptográfico que permite conectarse a otro equipo de forma segura, incluso a través de una red insegura como Internet. Su función principal es ofrecer un canal seguro para la administración remota de sistemas, la transferencia de archivos y la ejecución de comandos a distancia. Gracias a este protocolo, la información transmitida no es legible por terceras personas.

Características principales

  • Utiliza cifrado para proteger los datos transmitidos.
  • Evita el robo de contraseñas y ataques de intermediarios (man-in-the-middle).

Cómo funciona

SSH se basa en un modelo cliente-servidor:

  • El cliente SSH inicia la conexión desde tu equipo local. Es la aplicación que utilizas para conectarte a un servidor remoto.
  • El servidor SSH escucha conexiones entrantes y valida la identidad del usuario. Se ejecuta en el servidor remoto al que deseas acceder.
  • Una vez autenticado, se establece un canal cifrado y se pueden ejecutar comandos o transferir archivos.

El protocolo SSH siempre es seguro!

Tipos de cifrado

  • Cifrado simétrico: En esta forma de cifrado se utiliza una clave secreta tanto para cifrado como el descifrado del mensaje. Cualquiera que tenga el mensaje, puede descifrar el mensaje. A menudo se llama clave compartidao shared key.
  • Cifrado asimétrico: En este tipo de cifrado se utilizan dos claves. una pública y una privada. Se hace uso de una clave para el cifrado, y de la otra para el descifrado. Cuando un cliente se conecta a un servidor, utiliza la clave pública del servidor para cifrar un mensaje que sólo puede descifrarse con la clave privada correspondiente.
