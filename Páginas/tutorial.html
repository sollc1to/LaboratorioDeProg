¿Qué es SSH? SSH (Secure Shell) es un protocolo que te permite conectarte de forma segura a otro equipo a través de una red.

📋 Activación por Sistema Operativo

🐧 En Linux (Ubuntu/Debian)

Instalar el servidor SSH:

bash
sudo apt update
sudo apt install openssh-server

Verificar que este funcionando:

bash
sudo systemctl status ssh

Si no esta activo, iniciarlo:

bash
sudo systemctl start ssh
sudo systemctl enable ssh

🔒 Configuración básica de seguridad

Cambiar el puerto predeterminado:

bash
sudo nano /etc/ssh/sshd_config
# Cambiar: Port 22 → Port 2222

Reiniciar el servicio:

bash
sudo systemctl restart ssh

🔑 Cómo conectarse

bash
ssh usuario@direccion-ip -p puerto
# Ejemplo: ssh miusuario@192.168.1.100 -p 22

📋 Verificación final

bash
sudo netstat -tulpn | grep :22
# Para confirmar que todo funciona:
🐧

¡Configuración SSH completada! 🎉
